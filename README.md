# C Programming Project

## 📌 Description

This project is written in the **C programming language**.
It demonstrates basic concepts of C such as variables, loops, functions, arrays, and file handling.

The goal of this project is to practice and understand fundamental programming concepts using C.

## ⚙️ Requirements

* GCC Compiler
* Any C IDE or Code Editor (VS Code, CodeBlocks, Dev-C++)
* Operating System: Windows / Linux / macOS

##

## ▶️ How to Compile

Use the following command in the terminal:

```
gcc main.c -o program
```

## ▶️ How to Run

After compiling, run the program using:

```
./program
```

(Windows users can run `program.exe`)

## 📖 Concepts Used

* Variables and Data Types
* Conditional Statements (if, switch)
* Loops (for, while, do-while)
* Functions
* Arrays
* Pointers
* File Handling

## 🎯 Purpose

This repository is created for learning and practicing **C programming** and improving problem-solving skills.
